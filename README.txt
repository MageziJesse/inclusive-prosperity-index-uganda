# Telecom Construction Daily Reports System

This is a simple browser-based prototype for managing telecommunications construction site reports.

## Features
- Construction sites displayed on a map
- Site summary showing completion percentage and cost so far
- Daily reports table
- Form for supervisors/site workers to submit daily reports
- Data saved in the browser using localStorage
- Demo sites for Kampala, Mukono, Jinja, and Mbarara

## How to Open
1. Extract the ZIP file.
2. Open `index.html` in a browser.
3. Click site markers on the map.
4. Click `+ Add Daily Report` to submit a report.
